# AXIOM — Web Autonomous App Builder Agent

> The open-source autonomous agent that thinks, plans, codes, tests, and deploys web apps — without you lifting a finger.

## 🔥 What is AXIOM?

AXIOM is an autonomous AI agent that takes a single prompt like **"Build me a SaaS todo app"** and:
1. 🧠 Analyzes & understands your goal
2. 📋 Writes a detailed build plan
3. 🏗️ Scaffolds the full project structure
4. 💻 Writes all the code
5. 🧪 Tests and fixes errors automatically
6. 🚀 Deploys the live web app

## 🛠️ Tech Stack

- **Brain:** GPT-4o / Claude 3.5 + LangGraph ReAct loop
- **Browser:** Playwright (visual feedback loop)
- **Code Execution:** Docker sandbox
- **Memory:** ChromaDB + PostgreSQL
- **Frontend:** React + TailwindCSS
- **Backend:** FastAPI (Python)
- **Deployment:** Railway / Vercel
